## 기술 목록

### HTML, CSS, Javascript

- 기본적인 문법과 Jquery에 대해 알고 있습니다.
- 기본적인 layout 방법(float, flex, grid)에 대해 알고 있습니다.
- Sass에 대한 경험이 있습니다.
- Nodejs(express) 환경에서의 서버(api) 개발도 익숙합니다.

### React,Vue

- 함수형 컴포넌트를 이용할 줄 알고, hook을 사용할 수 있습니다.
- graphQl,Apollo 사용 경험이 있습니다.
- Redux, Vuex 등 상태 관리 라이브러리를 개발 한 경험이 있습니다.
- Vue에 Mobx, Vuex 등을 이용해서 프로토 타입을 만든 경험이 있습니다.

### Java / Spring

- Spring을 이용한 프로젝트 경험이 있습니다.
- 어노테이션 AOC, DI 의 개념을 알고 활용 합니다.
- 처음 부터 끝까지 프로젝트 환경 구성을 해서 개발을 진행 한 경험이 있습니다.
- Spring Boot를 이용한 API 서버를 만든 경험이 있습니다.

### DB

- MS-SQL,Oracle,Mysql 등을 모두 사용해 보았습니다.
- 쿼리플랜을 보면서 개발 하는걸 지향합니다.
- 테이블간 관계를 보고 쉽게 Select 할 수 있습니다.
- Where절 이용시 묵시적 형 변환이 이루어져 느려지는 쿼리의 문제점을 생각하며 쿼리를 개발합니다.
- Join 절을 비교적 자유롭게 활용할 수 있으며 원하는 데이터를 추출 할 수 있습니다.
- (Oracle) 세션 모니터링을 이용해 죽지 않는 프로세스를 보고 관리 할 수 있습니다.
- (Oracle) 프로시저와 사용자 정의 함수 등을 개발 할 수 있으며 커서의 위험성을 알고 있습니다.

- mongoDB(mongoose)의 경험이 있습니다.

- 리눅스 서버 웹 운영 환경 셋팅 경험
- 각종 DB 툴 (Orange, Toad,Robo3T) 사용 경험
